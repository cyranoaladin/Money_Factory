## 1. AEPO Signals (Individual Pathways)
**Context:** Appears when a user completes a module, submits a task, or updates their profile.
**Trigger:** Real-time analysis of user state.

### UI Component: Toast / Floating Tooltip
*   **Icon:** Brain / Synapse (Lucide: `brain-circuit` or custom)

*   **Style:**

    *   Background: Dark glass (`bg-background-secondary/95`)
    *   Animation: Slide in from bottom-right.

## 2. AECO Signals (Cohort Orchestration)
**Context:** Appears during group tasks, bootcamp phases, or when sync milestones are reached.
**Trigger:** Collective state updates (e.g., "5/10 cohort members completed Task A").

### UI Component: Toast / Floating Tooltip
*   **Icon:** Users / Group (Lucide: `users` or custom)
*   **Title:** "AECO Sync"
*   **Message:** "Cohort Milestone detected. Synchronizing group objectives..."
*   **Style:**
    *   Border: Left-accent border (`border-solana-purple`)
    *   Background: Dark glass (`bg-background-secondary/95`)
    *   Animation: Slide in from bottom-right (stacked above AEPO if simultaneous).
